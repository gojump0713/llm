# AI 소프트웨어 공공조달 마켓 인텔리전스 대시보드

나라장터 **특정품목 조달내역(인공지능(AI)소프트웨어)** 엑셀을 분석하는 단일 페이지 인터랙티브 대시보드입니다.
모든 처리는 브라우저(클라이언트)에서 이뤄지며 외부 API 호출이 없습니다.

## 주요 기능

- **KPI** 4종: 공급기업 수 · 총 계약금액 · 총 계약 건수 · 평균 계약 단가 (count-up + 직전 동기간 대비 증감)
- **공급기업 랭킹** TOP 15 (금액/건수 토글, 막대 클릭 → 필터)
- **월별 조달 추이** 콤보 차트 (막대=금액, 라인=건수, 브러시로 기간 선택)
- **수요기관 TOP 10** (막대 클릭 → 계약명·공급기업 드릴다운)
- **품목/서비스별 비중** 도넛 (소관구분/기업구분/계약방법 토글, 조각 클릭 → 필터)
- **AI 소프트웨어 소개** 카드 그리드 (검색·정렬)
- **원본 데이터 테이블** (정렬/검색/페이지네이션/CSV 내보내기)
- **크로스 필터링**: 모든 필터·차트 클릭이 전역 상태에 반영되어 동시 갱신, 활성 필터 칩·초기화
- **엑셀 업로드**: 동일 형식의 엑셀을 올리면 즉시 재분석 (헤더명 기반 자동 매핑)
- 다크/라이트 테마 토글, 데스크탑 그리드 ↔ 모바일 스택 반응형

## 기술 스택

React 18 · Vite 6 · Tailwind CSS v4 · Recharts · SheetJS(xlsx)

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # dist/index.html 단일 파일 빌드 (누구나 파일 하나로 실행)
npm run preview  # 빌드 결과 미리보기
```

> `npm run build` 결과물 `dist/index.html` 은 JS/CSS/샘플데이터가 모두 인라인된
> **자체 완결 단일 파일**이라, 별도 서버 없이 더블클릭으로 열어도 동작합니다.

## 데이터 처리 규칙

- 헤더는 헤더명(`계약(납품요구)일자` 등)으로 자동 탐지 → 컬럼 순서가 달라도 안전
- 금액: 원 단위 → 화면은 억/만 단위 가독 포맷 (예: `19.1억`, `7,700만`)
- 계약 건수: 기본 **계약(납품요구)번호 고유 기준**, 헤더 토글로 라인 건수
- 품목명을 콤마로 분해해 업체/제품/설명 추출, 의미정보가 없으면 계약명으로 보완
- 빈 금액/날짜 행 제외

## 구조

```
src/
├─ App.jsx                # 레이아웃 + 샘플 로드
├─ state/store.jsx        # 전역 상태(필터/토글/테마), 크로스필터링
├─ lib/
│  ├─ parse.js            # SheetJS 파싱 + 정제
│  ├─ aggregate.js        # 필터/집계 셀렉터
│  └─ format.js           # 금액/날짜 포맷
├─ components/            # Header, FilterBar, KpiRow, 차트, 카드, 테이블
└─ data/sample.js         # 내장 샘플(엑셀 base64) — 업로드와 동일 파이프라인
```
